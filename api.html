DataPipes#

Iterable-style DataPipes for geospatial raster and vector data.

Based on https://github.com/pytorch/data/blob/v0.3.0/torchdata/datapipes/iter/load/online.py#L29-L59

class zen3geo.datapipes.RioXarrayReaderIterDataPipe(source_datapipe: torch.utils.data.dataset.IterDataPipe[str], **kwargs: Optional[Dict[str, Any]])[source]#

Takes raster files (e.g. GeoTIFFs) from local disk or URLs (as long as they can be read by rioxarray and/or rasterio) and yields tuples of filename and xarray.DataArray objects (functional name: read_from_rioxarray).

Parameters

  • source_datapipe (IterDataPipe[str]) ‚Äì A DataPipe that contains filepaths or URL links to raster files such as GeoTIFFs.

  • kwargs (Optional) ‚Äì Extra keyword arguments to pass to rioxarray.open_rasterio and/or rasterio.open. See https://corteva.github.io/rioxarray/stable/rioxarray.html#rioxarray-open-rasterio and https://rasterio.readthedocs.io/en/stable/api/rasterio.html#rasterio.open

Yields

stream_obj (Tuple[str, xarray.DataArray]) ‚Äì A tuple consisting of the filename that was passed in, and an xarray.DataArray object containing the raster data.

Example

>>> from torchdata.datapipes.iter import IterableWrapper
>>> from zen3geo import RioXarrayReader
...
>>> # Read in GeoTIFF data using DataPipe
>>> file_url: str = "https://github.com/GenericMappingTools/gmtserver-admin/raw/master/cache/earth_day_HD.tif"
>>> dp = IterableWrapper(iterable=[file_url])
>>> dp_rioxarray = dp.read_from_rioxarray()
...
>>> # Loop or iterate over the DataPipe stream
>>> it = iter(dp_rioxarray)
>>> filename, dataarray = next(it)
>>> filename
'https://github.com/GenericMappingTools/gmtserver-admin/raw/master/cache/earth_day_HD.tif'
